## 🚀 Visão Geral
Neste projeto foi desenvolvido um Dashboard Financeiro completo no Power BI, utilizando uma base de dados em Excel com todo o processo de tratamento e transformação realizado no Power Query.

O principal objetivo da solução é automatizar análises financeiras e eliminar processos manuais repetitivos. Dessa forma, sempre que a base de dados for atualizada, basta um clique em **Atualizar** para que todos os indicadores, gráficos e métricas sejam recalculados automaticamente.

---

# 📂 Fonte de Dados
📄 Base única em Excel contendo informações financeiras, como:

- Receitas
- Custos
- Despesas
- Vendas
- Metas comerciais

---

# 🎯 Objetivo do Dashboard
O dashboard foi desenvolvido para proporcionar uma visão estratégica da performance financeira da empresa, permitindo análises como:

- 💰 Faturamento total
- 💸 Produtos mais vendidos
- 🧾 Ranking de vendedores
- 📈 Acompanhamento de metas
- 👥 Receita por período
- 📦 Quantidade de produtos vendidos

---

# ⚙️ Funcionalidades do Dashboard
A solução permite ao usuário:

- 📅 Filtrar informações por ano através de segmentação dropdown
- 📊 Visualizar KPIs financeiros consolidados
- 📈 Analisar a evolução temporal de receitas, custos e despesas
- 👥 Explorar o desempenho de vendedores
- 📦 Monitorar produtos com maior volume de vendas
- 🔄 Atualizar automaticamente todos os dados com apenas um clique

💡 Todo o processo foi desenvolvido com foco em automação, produtividade e tomada de decisão baseada em dados.

---

# 📊 Visuais Desenvolvidos

- 📌 KPI de Faturamento
- 📌 KPI de Meta
- 🍩 Receita por Categoria (gráfico de rosca)
- 📈 Receita por Mês/Ano (linha e área)
- 📊 Receita por Cliente (barras horizontais)
- 📉 Custos e Despesas (colunas empilhadas)
- 🎯 Filtro de Ano (segmentação dropdown)

---

# 🔄 Processo de Transformação de Dados
Os dados foram tratados no Power Query utilizando processos de ETL, incluindo:

- Padronização de colunas
- Definição correta de tipos de dados
- Tratamento de inconsistências
- Limpeza automatizada de informações
- Criação de regras de transformação

💡 Todas as etapas de tratamento são reaplicadas automaticamente a cada atualização da base de dados.

---

# 🧠 Modelagem e Cálculos (DAX)
Foram desenvolvidas medidas e cálculos em DAX para garantir análises dinâmicas e indicadores estratégicos, possibilitando uma visualização mais inteligente e interativa dos dados financeiros.
